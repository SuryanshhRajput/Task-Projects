# Task 1 - Learning Project

This is my first project task as part of my learning journey in web development.

## Project Overview
This project is a simple landing page layout created to practice HTML and CSS fundamentals. The goal was to build a clean and attractive page using basic layout techniques.

## What I Learned
- How to structure a webpage using HTML
- How to style elements with CSS
- How to use Flexbox for layout
- How to use positioning for alignment and design
- How to create a simple responsive layout concept

## Techniques Used
In this project, I used only:
- Flexbox
- Positioning

These techniques were used to arrange the navbar, content sections, images, and other UI elements.

## Project Files
- index.html - Contains the webpage structure
- style.css - Contains all the styling for the page
- images/ - Contains the images used in the project

## Purpose
This project is a beginner-friendly task where I practiced building a layout from scratch and improved my understanding of core CSS concepts.

## Notes
This is a learning project, and the design is simple. The main focus was to understand and apply layout techniques practically.
