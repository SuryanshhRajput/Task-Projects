# Responsive Assignment

## Preview
https://suryansh-singh-responsive-landing-p.vercel.app/

## Overview
This project is a responsive landing page built from a mobile-first design and adapted to desktop screens using CSS layout techniques.

## How it was made
- Built with `HTML` and `CSS` only.
- Designed to match a provided mobile footer layout and then extended to the desktop view.
- Used responsive design strategies to preserve the same layout and styling across devices.
- Implemented a horizontal card slider in `section-8` using `flexbox`, `overflow-x`, and `scroll-snap`.

## Tech used
- HTML5
- CSS3
- Flexbox for layout and responsive row/column structures
- Media queries for mobile/desktop breakpoints

## Features
- Fully responsive landing page structure
- Mobile-first footer design that was preserved and replicated for desktop
- Horizontal card slider behavior in `section-8` with visible peek of the next card
- Desktop footer updated to match the mobile footer style
- Input and button layout optimized for mobile without breaking desktop view

## Notes
- The result is a modern landing page with a reference mobile footer and a desktop version that mirrors the same styling.
